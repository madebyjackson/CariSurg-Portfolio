## Getting Started

### Prerequisites

- Python 3.10+
- Jupyter Notebook / JupyterLab
- Git

### Installation

Clone the repository:

```bash
git clone https://github.com/SupremeHimself/CariSurg-Portfolio.git
cd CariSurg-Portfolio
```

Create and activate a virtual environment:

```bash
python -m venv venv
```

Windows:

```bash
venv\Scripts\activate
```

Linux/macOS:

```bash
source venv/bin/activate
```

Install dependencies:

```bash
pip install -r requirements.txt
```

---

## Running Notebooks

Launch Jupyter:

```bash
jupyter notebook
```

or

```bash
jupyter lab
```

Navigate to the appropriate assignment folder and open the desired notebook.
